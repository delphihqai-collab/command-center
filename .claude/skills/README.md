# Skills

Reusable procedural expertise. Each skill is a focused, composable procedure that can be invoked across many prompts and tasks.

## Skills to Create (post-build)
- `supabase-migration.md` — how to write, validate, and apply a safe Supabase migration
- `component-creation.md` — how to create a new page or component in this project
- `agent-integration.md` — how to wire a new agent report type into the DB and the UI
- `rls-policy.md` — how to write and test a Row-Level Security policy
- `schema-update.md` — how to extend the DB schema without breaking existing data

## Do Not Create Skills Until Instructed
HERMES will analyse the completed codebase and extract the right skills from actual patterns used during the build.
